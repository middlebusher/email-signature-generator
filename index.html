How to install in Gmail

  1. Click Copy to Gmail above (copies the rendered signature, not the code).
  2. Open Gmail → Settings (gear icon) → See all settings.
  3. Scroll to the Signature section. Click Create new if you don't have one, or select your existing signature.
  4. Click inside the signature box and paste with ⌘V / Ctrl+V. Formatting will be preserved.
  5. Under Signature defaults, set this signature for new mails and replies.
  6. Scroll to the bottom and click Save changes. Send yourself a test email to confirm.
